Tab 1 — Entry Point Redesign
Add "Include Payment History" option to Proposal Download
Wherever users access their proposal, the existing Download PDF text link becomes a dropdown with two options: Proposal Only (the existing behavior) or Include Payment History (the proposal PDF + an appended Payment History page).
User side only. Both views below are the user side (the contractor/sales opening their own proposal). The proposal opened from their client slide will NOT show this download option — clients should not have visibility into payment information. Click the Download PDF button in each view to see the dropdown.

Tab 2 — PDF Output Preview
Append a Payment History page to the proposal PDF
When users select Include Payment History on download, this page is appended directly after the proposal PDF on a US Letter 816 × 1056 sheet. It contains an Invoice Schedule and Payment Records reflecting the contract's payment activity. Switch State to see different points in the contract lifecycle; the last tab ★ Status Variants demonstrates every visual variant in one view for engineering reference.
